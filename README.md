# Netflix Data Visualization and Storytelling

## Project Overview

This project explores the Netflix Movies and TV Shows dataset using Python visualization libraries. The goal is to uncover meaningful insights and communicate them effectively through visual storytelling.

---

## Objectives

- Perform exploratory data analysis
- Clean the dataset
- Create compelling visualizations
- Build a narrative from the data
- Present key findings

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud
- Jupyter Notebook

---

## Visualizations

- Content Growth
- Movies vs TV Shows
- Top Countries
- Ratings Distribution
- Genre Distribution
- Movie Duration
- Release Year Distribution
- Description Word Cloud

---

## Key Insights

- Netflix experienced rapid growth after 2015.
- Movies dominate the platform.
- The United States is the leading content producer.
- Drama is the most common genre.
- Most movies last between 80 and 120 minutes.
- Mature audience ratings are the most frequent.
